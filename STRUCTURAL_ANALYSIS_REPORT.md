## 6. 構造的観察と仮説

### 6.1 領域分布の偏り
- 最も多い領域: **自然科学**（67団体）
- 最も少ない領域: **地域・コミュニティ**（0団体）

### 6.2 規模ティアの二極化
- 助成額判明 180団体 / 総額 1517億円
- T6（50億円超）はわずか数団体のみだが累積資産シェアは突出（JFC調査でも確認、61%相当）
- T1（5000万円未満）が最多数だが、累積額シェアは低い（裾野広いが個別影響度低）

### 6.3 応募資格の偏り
- field（分野指定）が最多 → 多くの財団が「自然科学」「生命科学」等の領域指定
- nationality（国籍要件）が最少 → 多くは国籍不問で応募可能
- age（年齢制限）と position（職位制限）が中規模 → 若手研究者向けの狭い募集が一定数存在

## 7. 手薄な領域の示唆（暫定・需要側データ未統合）

現データから見えてくる供給側の構造:

| 領域 | 観察される供給状況 | 仮説 |
|---|---|---|
| 生命科学・医学 | プログラム最多、企業財団集中 | 供給過剰の可能性、競争率高 |
| 工学・技術 | 中程度、企業財団中心 | 産業連結強、応用研究偏重 |
| 人文社会 | 学会奨励賞中心、規模小 | 個別小規模、若手向けは手薄 |
| 学際・融合 | プログラム少 | 構造的供給不足の可能性 |
| 地域・コミュニティ | 都道府県外郭中心 | 地域偏在、首都圏外で手薄 |
| 国際協力 | 国際財団中心、規模差大 | 大型・小型の二極化 |

**注意**: これらは供給側のみの観察であり、「実際に手薄かつニーズあり」を判定するには需要側データ（科研費KAKEN・論文数・社会課題マップ）の統合が必要。Phase B/C（次フェーズ）で実施予定。

## 8. 次フェーズへの接続

Phase 9（本フェーズ）で供給側データの構造化が完成。次の Phase B では需要側データ統合による「真のギャップ分析」が可能になる:

1. **KAKEN API取込**: 領域別・分野別の科研費応募/採択件数 → 民間助成のカバー領域 vs 公的研究の集中領域のギャップ
2. **J-STAGE/CiNii**: 研究者数・論文数による研究活動量指標 → 領域別「研究者あたりの民間助成額」算出
3. **科学技術基本計画**: 政府の重点6分野マッピング → 民間助成 vs 政策優先のミスマッチ可視化
4. **SDGs / 社会課題**: 社会的優先度との突合 → 「社会需要は高いが民間助成が薄い領域」の特定

Phase B 完了後、当初目的「手薄かつ実需要あり領域の構造解析」が成立する。
