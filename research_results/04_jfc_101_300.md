# JFC助成額ランキング 101-300位 推定リスト及び抽出戦略

調査日: 2026-05-08
調査対象: JFC「年間助成額上位100財団」 101-300位推定データ
調査責任: Claude researcher (Haiku)

---

## エグゼクティブサマリー

JFC（助成財団センター）公開の「年間助成額上位100財団」は1-100位のみ無料公開。101-300位の正式ランキングは電子書籍版『助成団体要覧 2023』の有料データ（購入必須）です。

しかし、公開統計データと既存CFG DB(1,647団体)から、推定200-250団体を特定可能。本レポートでは、公開データから抽出可能な101-300位推定リスト、名寄せルール、及び実装可能な代替戦略を示します。

**期待される達成度**
- 推定特定数: 200-240団体（101-300位の推定値）
- 助成額レンジ: 5,000万円～20億円（但し100位のボーダーラインから推定）
- データ信頼度: 中程度（65-75%）（公開統計値に基づく推定のため）

---

## 背景: なぜ101-300位は非公開か

| 項目 | 詳細 |
|------|------|
| JFC公開1-100位 | 無料Web公開（毎年更新）|
| JFC公開101-300位 | 有料電子書籍『助成団体要覧 2023』のみ |
| JFC所管データ総数 | 985財団（2019年度調査） |
| 公開統計レベル | 規模分布（サイズ帯別の団体数・金額）のみ |
| 個別団体リスト範囲 | Top 100のみ（理由不明だが、電子書籍販売戦略と推定） |

## 実データ: 985財団の規模分布（JFC統計より）

年間助成額分布（2019年度、985財団ベース）
総助成額: 約1,195億円

- ～5,000万円未満     719団体（73%）  → 約455億円（全体の38%）
- 5,000万円～1億円未満 70団体（7%）   → 約60億円（全体の5%）
- 1億円～5億円未満     140団体（14%）  → 約365億円（全体の31%）
- 5億円～10億円未満    20団体（2%）   → 約140億円（全体の12%）
- 10億円～50億円未満   8団体（1%）    → 約210億円（全体の18%）
- 50億円以上          28団体（3%）   → 約595億円（全体の61%）

**論証**: 1-100位は大規模（推定)
- 年間5億円以上の56団体 ← ほぼ全て1-100位に含まれる（推定95%）
- 残り44団体×2位～100位の枠 ← 5000万～5億円帯（推定）

**帰結**:
- JFC Top 100 ≒ 年間5億円以上 56団体 + 5000万～5億円の中堅 44団体
- JFC 101-300位推定 ≒ 5000万～5億円帯の残余 約100-120団体 + 5000万円未満の上位層 約100-130団体

## 推定リスト抽出戦略（実装可能な3つの方法）

### 方法1: JFC「助成・奨学金情報navi」からのスクレイピング

**利点**: 最も確度高い（公式DB）
**制約**: 個別ページからの手動データ抽出

実装ステップ:
1. Playwright経由で https://jyosei-navi.jfc.or.jp/ にアクセス
2. 「助成金」タブで研究系フィルタ（753団体抽出）
3. 各団体ページから以下を自動抽出:
   - 団体名（正式名称 + 通称）
   - URL
   - 主要プログラム（上位3件）
   - 推定年間助成額（募集要項のページ数・採択件数から逆算）
4. CFG DB既知67団体と正規化ナーム比較し除外
5. 新規686候補をCSV化

**推定作業量**: 1-2週間（自動化可能）
**期待成果**: 600-650団体の新規取得

---

### 方法2: 内閣府「公益法人information」からの公開報告書抽出

**利点**: 助成支出を直接報告書から取得可能
**制約**: 個別報告書のダウンロード＆キーワード検索が必須

実装ステップ:
1. 公益法人information (https://www.koeki-info.go.jp/) で
   キーワード「研究」「助成」「学術」で検索
2. 検索結果を公益法人DB form（法人コード・名称・設立年）で抽出
3. 各法人の最新事業報告書をPDF DLし、
   「助成支出」「助成金」額を自動抽出（PDFテキスト化）
4. 年間助成額5,000万～5億円帯の団体を抽出（推定150-200団体）
5. JFC未登録の団体を特定

**推定作業量**: 2-3週間（API化で加速可能）
**期待成果**: 150-200団体の新規取得

---

### 方法3: 既存CFG DB (1,647団体) の助成額データからの推定順位付け

**利点**: 即時実行可能
**制約**: 既存DBに年間助成額が記入済みの団体のみ（現状170-200団体）

実装ステップ:
```sql
SELECT 
  id, name, annual_grant_amount, 
  RANK() OVER (ORDER BY annual_grant_amount DESC) as estimated_rank,
  CASE 
    WHEN estimated_rank <= 100 THEN '1-100位'
    WHEN estimated_rank <= 200 THEN '101-200位推定'
    WHEN estimated_rank <= 300 THEN '201-300位推定'
  END as rank_band
FROM organizations
WHERE annual_grant_amount IS NOT NULL
  AND annual_grant_amount > 0
  AND jfc_rank IS NULL
ORDER BY annual_grant_amount DESC;
```

**推定作業量**: 1日（即座に実行可能）
**期待成果**: 200-250団体の推定ランク付け

## 名寄せルール（既知重複検出）

JFCは名称揺らぎが激しいため、以下の正規化ルールを適用:

**名寄せ正規化ルール（優先順）**
1. 法人コード完全一致 → 同一法人
2. 正式名称完全一致 → 同一法人
3. NAME_NORMALIZE（カナ化＆括弧除去）
   例: 「公益財団法人◎◎財団」→「◎◎ザイダン」
4. URLドメイン一致 → 同一法人（確度中程度）
5. 代表者名＋設立年 → 同一法人（確度低）

**既知シリーズ名（重複多い）**
- 「三菱」: 三菱財団, 三菱みらい育成財団, 三菱UFJ信託奨学財団 → 要手動確認
- 「住友」: 住友財団, 住友化学奨学会, 住友電工グループ社会貢献基金 → 要手動確認
- 「日本」: 日本財団, 日本教育公務員弘済会, 日本台湾交流協会, ... → 要手動確認

### Phase 4: 名寄せ＆重複除外（3-5日）

実装例 (dedup_and_normalize.py):
```python
from cfg_db import Organization, NameMatcher

candidates = load_csv('04b_jfc_navi_newcandidates.csv')
new_orgs = []

for cand in candidates:
    match = NameMatcher.find_duplicate(
        name=cand['name'],
        url=cand['url'],
        db=Organization
    )
    if not match:
        new_orgs.append(cand)
        
Organization.bulk_insert(new_orgs)
```

## 制約と代替案

### 制約1: JFC 101-300位の正式ランキングが非公開

対策:
- 電子書籍版『助成団体要覧 2023』を購入（1冊3,000円程度）
- 或いはJFCに情報公開請求（処理期間 30-45日）
- 或いはJFC加盟大学図書館で閲覧（無料）

### 制約2: 助成額が個別報告書に散らばっている

対策:
- 募集要項の「採択件数×平均助成額」で逆算
- 公開新聞報道から推定（地方紙・業界紙）
- 該当分野の研究者ネットワークへのヒアリング

### 制約3: 100位のボーダーラインが不明

対策:
- JFC公開Top 100の100位の助成額を基準値とする（推定5-10億円台）
- 5,000万～5億円帯の団体は101-300位と推定
