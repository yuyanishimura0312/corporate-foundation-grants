## 2. 4スプリント実行計画

### Sprint 1: 自動抽出（既存データから、1-2日）

#### S1.1 採択予定数の逆算（高優先・低コスト）
- **方法**: `grant_results` から `(call_id, fiscal_year)` 別に集計し `actual_awards` に反映
- **追加抽出**: 過去3年の最頻値を `expected_awards` のデフォルト値として推定
- **期待**: 採択予定/実績データ 0.7% → **80%**
- **スクリプト**: `scripts/backfill_award_counts.py`

#### S1.2 キーワード自動抽出（採択者titleから）
- **方法**: 2,122件 `grant_results.project_title` から名詞句抽出（jieba/janome等）
- **集約**: `grant_calls.keywords` に上位5キーワードを格納
- **期待**: keywords 9.2% → **70%**
- **スクリプト**: `scripts/extract_keywords_from_titles.py`

#### S1.3 細粒度カテゴリ分類（11×3階層）
- **方法**: program name + description + purpose をルールベース+LLM分類
- **構造**: foundation_categories に level 2-3 を追加
  - 例: life_science → 医学、生命科学基礎、薬学、看護、再生医療、ゲノム、神経、感染症
  - 例: engineering → 機械、電気電子、情報、化学工学、土木建築、材料、エネルギー
- **期待**: 540 programs に細分類タグ（複数可）付与 100%
- **スクリプト**: `scripts/refine_program_categories.py`

#### S1.4 応募資格の構造化（description解析）
- **方法**: `eligibility_criteria` の自動付与（criterion_type別）
  - age: "45歳以下" / "若手" / "博士課程"
  - position: "教授" / "准教授" / "PI" / "ポスドク"
  - nationality: "日本国籍" / "国籍不問"
  - field: 領域指定の有無
  - affiliation_type: 大学/企業/NPO
- **対象**: `grant_programs.description` + `grant_calls.summary` から抽出
- **期待**: 応募資格 22% → **60%**
- **スクリプト**: `scripts/extract_eligibility.py`

**Sprint 1 完了時の到達点**:
- 採択数 80%、キーワード 70%、細分類 100%、応募資格 60%

### Sprint 2: 公益法人info財務PDF全自動巡回（3-5日）

#### S2.1 公益法人info全自動巡回
- **方法**: 既存 `extract_annual_reports.py` を全1,500団体（koeki_id持ち）に拡張
- **追加機能**:
  - 公益法人info内部リンクの自動辿り（事業報告→財務諸表PDF）
  - 「公益目的事業費」「助成事業費」「正味財産増減計算書」3パターン抽出
  - 過去5年分の年度別履歴を `annual_grant_amount_history` JSONに格納
- **対象**: 1,500団体のうち年間助成額判明していない約1,400団体
- **レート**: 1req/3sec、所要 ~70分
- **期待**: 年間助成額 6.9% → **30-40%**（800-840団体）
- **拡張点**: OCR対応（pdf2image+tesseract）で画像PDF対応

#### S2.2 大型財団の年次報告精緻化
- **対象**: JFC top81 + 親企業判明した825 corporate財団
- **方法**: 公式URL→「IR」「事業報告」「財務情報」リンク辿り
- **期待**: 中型財団の助成額判明を更に+200

**Sprint 2 完了時の到達点**:
- 年間助成額 6.9% → **40-50%**（840-1,050団体）
