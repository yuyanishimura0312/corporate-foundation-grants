# 研究助成財団DB（CFG）カバー率分析レポート

調査日: 2026-05-08（Phase 4完了時点）

## エグゼクティブサマリー

CFG収録財団数 **1,647団体** に対し、研究助成財団エコシステムを以下3レイヤーで再定義し、カバー率を推定する。

| 定義レイヤー | 母集団推定 | CFGカバー率推定 | ギャップ |
|---|---|---|---|
| **A. 厳密**（研究助成を主目的とする財団） | 600-800団体 | **80-100%** | ほぼ網羅、漏れは特殊用途のみ |
| **B. 標準**（公開DB登録済の研究関連助成団体） | 1,500-2,500団体 | **65-90%** | UMIN医学系・学会・地域財団 |
| **C. 広義**（研究助成を行う日本国内全団体） | 3,000-5,000団体 | **35-55%** | 業界団体・信託・地域・学会賞 |

CFGの強みは公益財団中心の構造化網羅、弱みは学会・医学系・地方の取りこぼし。

## 1. 母集団推定の根拠

### 1.1 公的データソース別

| データソース | 母集団数 | 内容 |
|---|---|---|
| JFC助成・奨学金情報navi | 2,539団体 / 8,091プログラム | 全助成形式（研究/奨学/留学/事業/表彰） |
| JFC研究助成特化版 | 753団体 / 1,213プログラム | 「助成金応募ガイド」掲載 |
| JFC法人会員 | 297団体 | JFC直接会員（質保証） |
| JFC 2019年調査統計 | 985団体 | 構造分析対象（規模分布算出基盤） |
| UMIN研究費補助機関DB | 618機関 | 医学・生命科学中心 |
| 内閣府公益法人info | 9,712法人 / 約3,000-4,000公益財団 | 全公益認定法人マスタ |
| CANPAN助成制度DB | ~1,000-1,500件 | NPO・市民活動寄り |

### 1.2 集合論的推定

ベン図的に統合すると以下と推定:
- **コア重複圏（複数DB登録）**: ~600-800団体 → 厳密定義
- **JFC + 内閣府研究系のユニオン**: ~1,500-2,000団体 → 標準定義
- **+ 学会賞 + 大学基金 + 地域 + 業界団体**: ~3,000-5,000団体 → 広義

### 1.3 質的検証

CFG収録1,647団体のうち、JFCランク付与済67団体は2022年度上位100の67%に相当。残り33%（鉄道弘済会・日本財団・JKA等）は規模大ながら名称揺れで未マッチ → ファジーマッチで98%回収可能（実装済）。

## 2. CFGの構造的強みと弱み

### 強み
1. **法人形態の網羅性**: 公益財団法人 1,102団体（85%）が中心、研究助成の中核プレーヤーをカバー
2. **設立者形態の多様性**: 企業財団825・個人財団221・学術系248・国際124・NPO190の8カテゴリで内訳化
3. **採択者データの先行**: 2,122件（武田1,665・三菱292・稲盛165）— 同類DBで先行する規模
4. **構造化スキーマ**: 14テーブル（grant_results・foundation_categories・parent_companies等）で深堀分析可能

### 弱み（カバー率向上の主要ターゲット）
| ギャップ領域 | 推定欠損数 | 理由 |
|---|---|---|
| **UMIN医学系補助機関** | +50-100 | 医学部系の機関助成が未統合 |
| **学会研究奨励賞** | +200-400 | 日本物理学会賞等、約200学会 × 平均1.5プログラム |
| **大学独自基金** | +50-100 | 京都大学基金・東大稲盛など、大学固有の研究助成 |
| **地域研究振興財団** | +50-150 | 都道府県レベルの研究振興財団・産業支援機関 |
| **業界団体研究助成** | +30-80 | 経団連・建設業協会・銀行協会等の研究助成プログラム |
| **国際財団日本窓口** | +20-50 | Templeton Japan・Sasakawa Peace等の国際財団 |
| **信託・保険系冠財団** | +30-50 | みずほ信託・三井住友信託・東京海上等 |
| **製薬・医療機器系** | +30-60 | アステラス・第一三共・武田薬品本体・テルモ等 |

合計新規候補: **+460-990団体**（広義カバー率を大幅改善）

## 3. カバー率向上ステップ（優先順位付き）

### Phase 5（即着手・1-2週間）

#### S1. JFC navi 2,539団体スクレイピング
- **手段**: Playwright/requests で `https://jyosei-navi.jfc.or.jp/` 全件巡回
- **対象**: 「研究助成」事業形式フィルタの全件 → 753団体（既存重複除外で +200-400新規）
- **必要工数**: 中（レート制限尊重で2-3日）
- **期待効果**: B層カバー率を 65-90% → 85-95% に押上

#### S2. UMIN医学系618機関統合
- **手段**: UMINデータベース https://center6.umin.ac.jp/ から研究費補助機関リスト抽出
- **対象**: 医学・生命科学領域の618機関（重複除外で +50-100新規）
- **期待効果**: 医学系カバー率を100%近くに

#### S3. URL自動探索の本格運用
- **手段**: 既存 `discover_urls.py` を1,071団体に展開、DuckDuckGo + 公益法人info内部リンク
- **期待効果**: URL判明 576 → 1,200-1,400団体（70-85%）

#### S4. 採択者収集の全Priority1財団展開
- **手段**: 旭硝子・住友・トヨタ・セコム・上原・鹿島・中谷・テルモの採択者パーサ追加
- **期待効果**: grant_results 2,122 → 5,000-7,000件

### Phase 6（中期・2-4週間）

#### M1. 学会研究奨励賞リスト構築
- **手段**: 日本学術会議の協力学術研究団体リスト約1,800団体から、研究奨励賞・学会賞を運営する団体200-400を抽出
- **対象**: 日本物理学会・日本化学会・電子情報通信学会等
- **新規スキーマ**: organization.foundation_subtype に 'academic_society' 追加検討

#### M2. 大学独自研究基金
- **手段**: 各大学のGrants Office推奨リスト（東大・京大・阪大・東北大・名大・北大・九大・東工大）から大学固有基金を抽出
- **対象**: 京都大学基金・東京大学EMP・大阪大学未来基金等

#### M3. 都道府県地域研究振興財団
- **手段**: 各都道府県の産業労働部所管財団リスト
- **対象**: 47都道府県 × 平均2-3団体 = 90-150団体（重複除外で +30-80）

### Phase 7（長期・1-2ヶ月）

#### L1. 国際財団・外資系日本法人
- **手段**: アメリカFoundation Directory Online等から日本人応募可能な財団を逆引き
- **対象**: Templeton Japan・Sasakawa Peace・Ford Foundation Japan等

#### L2. 業界団体研究助成
- **手段**: 経団連・日商・各業界団体の研究助成プログラム調査
- **対象**: 建設業協会・銀行協会・自動車工業会・電気事業連合会等

#### L3. 採択者ネットワーク分析章
- **手段**: 所属機関集中度・研究分野配分・若手比率・性別比のダッシュボード追加
- **期待効果**: DBが「検索道具」から「分析プラットフォーム」へ進化

## 5. 結論

CFGは**「厳密定義の研究助成財団」では既に高カバー率（80-100%）を達成**している一方、**学会・大学独自基金・地域財団のロングテール**で取りこぼしがある。次のステップは「個別追加」ではなく「機械的網羅」—JFC naviスクレイピング・UMIN統合・URL自動探索・パーサ拡張を組み合わせれば、3ヶ月で**広義カバー率85%以上**は到達可能と推定する。

実装可能性は高く、ボトルネックは**個別Web情報の正規化と重複検出ロジック**にある。Phase 4で既に実装済みの正規化・推論基盤（refine_other_v2・discover_urls・extract_annual_reports）は流用可能で、Phase 5以降は主にデータ収集の量的拡張となる。
